知识杂记

  1. jre系统库是什么
  • java runtime environment(Java运行环境)
  • 包含jvm, java核心类库和支持文件，不包含jdk, 编译器，调试器，其他工具
  1. java类名规范
  • java保存的文件名必须与类名一致
  • 如果文件中只有一个类，文件名必须与类名一致
  • 一个Java文件中只能有一个public类
  • 如果文件中不止一个类，文件名必须与public类名一致
  • 如果文件中不止一个类，而且没有public类，文件名可与任一类名一致
